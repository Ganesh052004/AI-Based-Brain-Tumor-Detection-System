# Brain Tumor Detection System

A web-based Brain Tumor Detection System built using Python, Flask, HTML, CSS, JavaScript, and Deep Learning.  
This project predicts whether a brain MRI image contains a tumor or not using a trained AI model.

---

## Features

- Brain MRI Image Upload
- AI-based Tumor Prediction
- User Authentication System
- Admin Dashboard
- Health Tips Section
- FAQ Management
- Responsive UI Design

---

Installation
1. Clone Repository
git clone https://github.com/YOUR_USERNAME/brain-tumor-detection.git
2. Open Project Folder
cd brain-tumor-detection
3. Install Dependencies
pip install flask tensorflow keras numpy pillow
4. Run Project
python app.py
